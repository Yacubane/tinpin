<div align="justify">
TinPin is an open-source gamepad project, which aims to create wireless motion game controller that will combine button and motion input to control various video games in a different than usual way.
</div>

## :camera: Presentation video
[![IMAGE ALT TEXT HERE](https://img.youtube.com/vi/T2ajyQ1guBs/0.jpg)](https://www.youtube.com/watch?v=T2ajyQ1guBs)

## :bulb: Features
* 9 buttons, Accelerometer, Gyroscope (6 axis)
* Battery
* Wireless connection to host
* Possibility to create custom mapping for games

## :book: Documentation
<div align="justify">
Documentation is available [here](https://1drv.ms/b/s!Ap19wJ6OUb8Y_C6AiVWxbAW8CylH?e=whoII0). It covers both parts of the controller (The motion controller itself and the controller dongle) in the way of hardware (used parts, schematics, building process) and software (calibration, indicators, communication, used libraries).
